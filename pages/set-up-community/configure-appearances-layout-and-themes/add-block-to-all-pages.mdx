# Add Block to All Pages with Layout Editor
This article describes how to add a block to all pages of your site using the Layout Editor feature in the Admin Control Panel (AdminCP). By following these steps, you can ensure that the block is consistently displayed across your entire site, providing a unified experience for your users.
1. Sign in to your site with an administrator account.
2. Open the **Header Account** menu on the top-right corner, click on the **Editor Mode** menu to enable the **Live Editor** mode. The toolbox will be displayed on the left side of the screen.
3. In the Layout Editor toolbox, click on the **Edit Site Content** option (the second option in the Edit section).
4. The page will switch to the site-wide editing mode, allowing you to add blocks that will appear on all pages of your site.
5. Click on the **Add Block** button in the desired section (e.g., Header, Top, Side, Main, Subside) where you want the block to be displayed.
6. In the block selection menu, choose the block you want to add from the list of available blocks. You can use the search function to quickly find the block you need.
7. After selecting the block, configure any necessary settings for the block, such as title, visibility, or specific content.
8. Once you have configured the block, click **Save** or **Save Changes** to add the block to all pages.
9. Publish the changes and rebuild the site to apply the new block across all pages.

Done! You have successfully added a block to all pages of your site using the Layout Editor. Users will now see the new block on every page they visit, providing a consistent experience throughout your site.
