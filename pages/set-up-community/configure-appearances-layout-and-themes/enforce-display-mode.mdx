# Configuring the Enforce Display Mode Setting

phpFox V5 (MetaFox) provides administrators with the ability to enforce a specific display mode (dark mode or light mode) across the site, ensuring a consistent user experience regardless of individual user preferences.

To force the use of a particular theme mode:

1. Access the Admin Control Panel (AdminCP) by logging in with administrative credentials.

2. Navigate to **Settings** > **Site Settings**.

3. Locate the **Enforce Display Mode** configuration option.

4. Select one of the available modes:
   - **Light Mode**: Enforces the light theme, preventing users from switching to dark mode.
   - **Dark Mode**: Enforces the dark theme, ensuring all users experience the dark interface.

5. Save the changes to apply the setting site-wide.

This configuration overrides any user-selected theme preferences and applies immediately to all site visitors.
