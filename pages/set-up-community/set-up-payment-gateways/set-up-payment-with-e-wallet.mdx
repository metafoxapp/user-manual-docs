# Set up payment with E-Wallet

In MetaFox, users can make payments with money in their E-Wallet to purchase listing items, subscription plan, activity points. This guide will guide you how to enable payment option with E-Wallet on your MetaFox site

Let's log into the AdminCP Dashboard. Then go the App Settings > Payments and activate E-Wallet option as the below screenshot

![Screenshot 2023-11-07 at 11.49.35.png](/images/Screenshot%202023-11-07%20at%2011.49.35.png)

The setup is done. Now you will see the E-Wall option showing up when making payments as below.

![Screenshot 2023-11-07 at 11.54.11.png](/images/Screenshot%202023-11-07%20at%2011.54.11.png)

Note that if your site supports multiple currencies and want to allow users to purchase with currencies different from USD, you will need to set up and enable Exchange Rates in **App Settings** > **E-Wallet** > **Exchange Rates** page.
