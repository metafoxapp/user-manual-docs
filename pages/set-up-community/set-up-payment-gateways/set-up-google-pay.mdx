# Set up Google Pay

This guide aims to assist you in seamlessly integrating Google Pay as a payment method on your MetaFox platform. WithÂ Google Pay, your users can make secure transactions using their Android devices, enhancing their payment experience.

Assume that you have installed the **Google Pay** app on your MetaFox successfully. You may want to refer the for app if needed.

Let's go to **AdminCP** > **App Settings** > **Payment**. You will see the Google Pay item listed as below

![Screenshot 2025-04-23 at 10.40.02.png](/images/Screenshot%202025-04-23%20at%2010.40.02.png)

Edit the Google Pay and choose a payment processor you want to use with Google Pay. You will need to make sure that the chosen Payment Processor has been set up properly. You can refer to the guide for more detail.

Then, Check the Active checkbox and save changes to complete the set up for Google Pay.

![Screenshot 2025-04-23 at 10.41.34.png](/images/Screenshot%202025-04-23%20at%2010.41.34.png)

Congratulations. You have done the setup for Google Pay on MetaFox site!
