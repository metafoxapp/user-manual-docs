# Configure ad placement for sponsored items

This article will guide you to configure ad placement for sponsored items. For example, you want to show sponsored blogs on the Video - Home page. You can follow the steps below to add an block into the Video Home  page and configure it to show sponsored blog items.

-   Log in to your MetaFox site with the Super Adminitrator or Admin user. Then, go to Video - Home page and enable **Editor Mode**. Choose **Edit Page Content** as we suppose to add a block to the Video Home page only. If you would like to add a block into all pages, you can use **Edit Site Content**.
-   Click on the **Plus** icon on the ***main*** section to add a block into this area, as we would like sponsored blogs to display in the main content of this page.
-   On the **Add Block** form, search *Browse Blogs* and click **Add Block** button.
-   Then, you can configure the Browse Blogs block as the below:
  - Add title: `Sponsored Blogs`
  -   In Container Type section, choose `Blog - Main Card` for Grid Item Style.
  -   In Data Source section, choose `/blog` for API URL and `view=sponsor&limit=3` for Sort & Filter. This configuration will make this block to show 3 sponsored blog items.
-   Then, click **Add** to add this block.
-   The last step is we will save changes for Editor Mode and Publish Layout to rebuild the frontend web.

Congratulation. We have done setup for a new Ads placement for sponsored items on the Video Home page. You can follow this procedure to add Ads placements for other pages of your MetaFox site.
