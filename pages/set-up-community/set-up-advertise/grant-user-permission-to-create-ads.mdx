# Grant user permission to create ads

This article will guide you to grant users permission to create ads on the MetaFox site. Assume that Advertise app has been installed correctly.

Let's log into the AdminCP. Then go to **App Settings** > **Advertise** > **Permissions**

![Screenshot 2023-07-05 at 09.59.34.png](/images/Screenshot%202023-07-05%20at%2009.59.34.png)

Select the **Role** that you want to grant permission on Advertise app.

In the image above, we have chosen the **Registered User** role. The permissions will be loaded for the chosen role automatically.

There are many permissions of the Advertise app that you can check and grant appropriate ones for that role by toggling associated settings

Once you have done configuration, click on **Save Changes** button at the bottom.

*Congratulation! You have granted permissions for a specific user role*
