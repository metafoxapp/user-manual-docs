# Set up backup

In this article, you will learn how to set up backup for your phpFox V5 platform. Regular backups are crucial for ensuring the safety and integrity of your data. By configuring a backup strategy, you can protect your community's information and quickly restore it in case of data loss or corruption.

phpFox V5 provides several options for backing up your data, including:

## Manual Backup with Backup Now in AdminCP.

Log in to your Admin Control Panel (AdminCP) with an administrator account, navigate to: **Maintenance** > **Backup** > **Backup Now** page. 

Then follow the prompts to create a backup of your database and files.

## Automated Backup using many options in AdminCP.

Log in to your Admin Control Panel (AdminCP) with an administrator account, navigate to: **Maintenance** > **Backup** > **Settings** page. 

Then enable site backup to automatically create backups of your site at regular intervals. You can choose the frequency of backups (e.g., daily, weekly) and specify the number of backup copies to retain.

DOne! You have successfully set up backup for your phpFox V5 platform. Regular backups will help ensure that your community's data is protected and can be restored in case of any issues.
