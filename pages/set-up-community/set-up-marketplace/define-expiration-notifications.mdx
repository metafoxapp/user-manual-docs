# Define Expiration Notifications for Marketplace Listings

To set the number of days before listing owners receive expiration notifications for their Marketplace listings, you can configure this setting from the Admin Control Panel (AdminCP). This allows you to notify sellers in advance when their listings are approaching expiration, giving them the opportunity to renew or update their listings as needed. Follow these steps to define expiration notifications for Marketplace listings:
1. Sign in to AdminCP with an administrator account.
2. Navigate to: App Settings > Marketplace > Settings
3. In the Settings view, locate the field labeled **Days to Notify Expiring Listing**.
4. Enter the desired number of days before expiration that you want the system to send notifications to listing owners.
5. Click **Save** to apply the changes.
6. Clear site caches to ensure that the new notification settings take effect on the frontend.  

Done! You have successfully defined expiration notifications for Marketplace listings. Listing owners will now receive notifications based on the number of days you specified before their listings expire, allowing them to take appropriate action to maintain their listings in the Marketplace.
