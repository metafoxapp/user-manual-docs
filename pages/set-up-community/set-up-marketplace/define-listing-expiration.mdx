# Define Listing Expiration for Marketplace

Administrators can set the number of days before listings in the Marketplace app expire. This allows you to manage the lifecycle of listings and ensure that outdated or inactive listings are removed from the Marketplace after a certain period. Follow these steps to define listing expiration for Marketplace:
1. Sign in to AdminCP with an administrator account.
2. Navigate to: App Settings > Marketplace > Settings 
3. In the Settings view, locate the field labeled **Days to Expire**.
4. Enter the desired number of days before listings expire in the Marketplace.
5. Click **Save** to apply the changes.
6. Clear site caches to ensure that the new expiration settings take effect on the frontend.

Done! You have successfully defined listing expiration for the Marketplace app. Listings will now expire based on the number of days you specified, helping to keep the Marketplace up-to-date and relevant for users.
