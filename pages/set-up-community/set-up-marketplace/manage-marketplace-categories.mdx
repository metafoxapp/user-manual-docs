# Manage Marketplace Categories

To organize listings effectively in the Marketplace app, you can create and manage categories from the Admin Control Panel (AdminCP). Categories help users find relevant listings more easily and allow sellers to classify their items appropriately. Follow these steps to manage categories for the Marketplace:
1. Sign in to AdminCP with an administrator account.
2. Navigate to: App Settings > Marketplace > Categories
3. In the Categories view, you can see a list of existing categories. To create a new category, click the **Add New Category** button.
4. Fill in the required information for the new category, such as the category name and description. You can also set a parent category if you want to create a subcategory.
5. After entering the details, click **Save** to create the new category.
6. To edit an existing category, click on the Option icon > Edit menu of that category in the list, make the necessary changes, and then click **Save**.
7. To delete a category, click on the Option icon > Delete menu of that category in the list, and confirm the deletion when prompted.

Done! You have successfully managed the categories for the Marketplace app. Users will now see the updated category structure when browsing listings, and sellers can classify their items under the appropriate categories.
