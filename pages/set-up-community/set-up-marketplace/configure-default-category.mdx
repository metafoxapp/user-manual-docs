# Configure Default Category for Marketplace

To set a default category for new listings in the Marketplace app, you can configure this setting from the Admin Control Panel (AdminCP). This ensures that when users create new listings, they will be automatically assigned to the default category unless they choose a different one. Follow these steps to configure the default category for the Marketplace:
1. Sign in to AdminCP with an administrator account.
2. Navigate to: App Settings > Marketplace > Categories
3. In the Categories view, locate the category you want to set as the default category for new listings.
4. Click on the Option icon > Mark as Default menu of that category in the list.

Done! You have successfully set a default category for new listings in the Marketplace app. When users create new listings, they will be automatically assigned to this default category unless they select a different one during the listing creation process.
