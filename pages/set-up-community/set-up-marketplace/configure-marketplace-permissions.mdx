# Configure Permissions for Roles on Marketplace

To control which user roles have access to specific features of the Marketplace app, you can configure permissions from the Admin Control Panel (AdminCP). This allows you to manage who can create listings, view certain pages, or perform administrative actions within the Marketplace. Follow these steps to set up permissions for different roles:
1. Sign in to AdminCP with an administrator account.
2. Navigate to: App Settings > Marketplace > Permissions
3. In the Permissions view, select certain user role to list the corresponding permissions for Marketplace features.
4. After configuring the permissions for the selected role, click **Save** to apply the changes.
5. Repeat steps 3-4 for each user role you want to configure permissions for.
6. Clear site caches to ensure that the new permissions take effect on the frontend.

Done! You have successfully configured permissions for different roles on the Marketplace app. Users will now have access to Marketplace features based on the permissions you set for their respective roles.
