# ChatPlus Server Requirements

ChatPlus server requires to be installed with **Docker** on dedicated server or VPS using Linux OS (such as Ubuntu, CentOS, Debian, etc). And, generally, the more powerful server is, the more users Chatplus can handle.

Here is our recommendation for the server configuration for Chatplus:

### FOR CHAT FEATURES

**Minimal Server Requirements**  
\- Single core (2 GHz)  
\- 1 GB RAM  
\- 30 GB of SSD  
The above minimal configuration is fine for small deployments of up to 200 users, up to 50 concurrently active and minimal level of mixed uploads, sharing, and chat activities.

\- Dual-core (2 GHz)  
\- 2 GB RAM  
\- 40 GB of SSD  
The above configuration can accommodate small deployments of up to 500 users, up to 100 concurrently active and moderate levels of mixed uploads, sharing, and chat activities.

### FOR AUDIO/VIDEO CALL FEATURES

**Recommended Server Requirements**  
\- Intel Xeon Platinum 8000 3.6 GHz x 4 core  
\- RAM: 8 GB  
\- Hardware basic: 250GB  
\- Network bandwidth : Min 250 Mbb/s

System requirement can serve 50 users with stream video, audio at the same time (default 720p)
