# Configure max file size by type

This article will guide you to configure maximum upload file size limits by file types. 

## Steps

- Go to AdminCP > Settings > Storages > Settings page. You will see settings to configure maximum upload file size limits by file types.

- Fill in the maximum file size limits for each file type (photo, video, music) in the corresponding fields. Then save the changes. 

Done! You have successfully configured maximum upload file size limits by file types. Users will not be able to upload files that exceed the configured limits for each file type.
