# Set up countries

This guide will walk you through the process of setting up countries in your phpFox V5, allowing you to manage and display country information effectively.

## Adding a New Country
1. Log in to the Admin Control Panel (AdminCP) with your administrator account.
2. Navigate to: **Localizations** >  **Countries**.
3. Click on the **Add New Country** button to create a new country entry.
4. Fill in the required fields:
   - **Name**: Enter the name of the country.
   - **Country ISO**: Provide the ISO code for the country (e.g., US for United States).   
5. Click **Save** to add the new country to the list.


## Editing an Existing Country
1. In the list of countries, locate the country you wish to edit.
2. Click on the **Edit** button next to the country entry.
3. Update the necessary fields (Name, Country ISO).
4. Click **Save** to apply the changes.

## Set default country
1. Navigate to: **Localizations** >  **Settings**.
2. In the settings page, find the **Default Country** option.
3. Select the desired default country from the dropdown menu.
4. Click **Save** to set the default country for your platform.

Countries are now set up and can be used in various features of your phpFox V5 site.
