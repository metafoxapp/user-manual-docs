# Set up Social Connect

Social Connect is a safe, fast, and easy way to log in by using existing accounts from Facebook, Google, etc. With Social Connect, saving time to sign up and securing passwords will make your website more friendly, professional, and interactive. Social Connect helps you reduce customer care and password recovery cost and achieve greater customer satisfaction by eliminating the frustration associated with user forgetting login credentials (usernames and passwords).

In this section, we will guide you to set up Social Login with Facebook, Google, and Apple
