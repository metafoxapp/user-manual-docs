# Import phrases from CSV file

Assume you have and done all modifications/translations on phrases of a specific language on your local machine. We will guide you to import phrases into the language. After importing, all phrases in your CSV file will be updated to the language.

Now, let's log into AdminCP. Then, go to **Localization** > **Languages**

Click on the **Option** icon of the language you want to import phrases to. Then, select the **Import phrases**.

![Screenshot 2023-07-03 at 10.27.45.png](/images/Screenshot%202023-07-03%20at%2010.27.45.png)

On the popup, select the CSV file on your local machine

![Screenshot 2023-07-03 at 10.28.12.png](/images/Screenshot%202023-07-03%20at%2010.28.12.png)

Click **Upload** button to start uploading CSV file. Phrases will be imported automatically when the upload is done.
