# Edit language phrase in AdminCP

This article will guide to edit or translate a language phrase of a certain language in AdminCP

Let's log into the AdminCP. Then, go to **Localization** > **Phrases**.

Search for the phrase you want to edit/translate. Please use some search options to help you filter phrases quickly. Then, click on the **Option** icon of that phrase, choose **Edit** option to edit

![Screenshot 2023-07-03 at 09.58.09.png](/images/Screenshot%202023-07-03%20at%2009.58.09.png)

  

Edit text on the popup. But please don't modify the variables (which are between the brackets symbols) since their values will be replaced dynamically when rendering.

![Screenshot 2023-07-03 at 09.58.39.png](/images/Screenshot%202023-07-03%20at%2009.58.39.png)

After completing your modification, click **Save Changes** button to update the phrase.

Last, you can  to apply changes of the phrase.
