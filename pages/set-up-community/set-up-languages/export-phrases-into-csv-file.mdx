# Export phrases into CSV file

This article will guide you to export phrases into CSV files so you can edit/translate all phrases more conveniently with your prefer editor tool

Let's log into AdminCP. Then, go to **Localization** > **Languages**.

Click on the **Option** icon > **Export phrases** option as below

![Screenshot 2023-07-03 at 10.16.00.png](/images/Screenshot%202023-07-03%20at%2010.16.00.png)

Choose the location and save the CSV file onto your machine. Now you can use editor supporting CSV format to edit/translate phrases on your local machine.
