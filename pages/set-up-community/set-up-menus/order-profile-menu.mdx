# Manage and Order Profile Menu Items

This article describes how to reorder menu items that appear on a user's profile page using the Admin Control Panel (AdminCP). Follow these steps to control the sequence and visibility of profile menu entries.


1. Sign in to AdminCP with an administrator account.
2. Go to: Appearances → Menus.
3. In the Menus view, open the filter for **Type** and select **Profile Menu**.
4. From the filtered list, locate and select the menu labeled `user.user.profileMenu` — this is the menu that controls items on the user profile page.
5. Click on `user.user.profileMenu` item to open the menu item list.
6. Reorder items using drag-and-drop handles or the provided up/down controls to set the desired sequence.
7. When the order is correct, click **Save** or **Save Changes** to persist the new order.
8. Clear site caches so the frontend reflects the update.
9. Verify the new order by visiting a user's profile page on the frontend.


Done! You have successfully reordered the profile menu items. Users will now see the updated menu sequence when they visit their profile pages.
