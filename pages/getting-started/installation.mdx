# Installation

This guide will walk you through installing phpFox on your server.

## System Requirements

Before installing phpFox, ensure your server meets the following requirements:

### Server Requirements
- **Web Server**: Apache 2.4+ or Nginx 1.18+
- **PHP**: 7.4 or higher (8.0+ recommended)
- **Database**: MySQL 5.7+ or MariaDB 10.2+
- **Memory**: Minimum 512MB RAM (2GB+ recommended)
- **Disk Space**: At least 500MB free space

### PHP Extensions Required
```bash
- mbstring
- mysqli or pdo_mysql
- curl
- gd or imagick
- xml
- zip
- json
- openssl
```

## Installation Methods

### Method 1: Automatic Installer (Recommended)

1. **Download phpFox**
   ```bash
   wget https://www.phpfox.com/download/latest.zip
   unzip latest.zip -d /var/www/phpfox
   ```

2. **Set Permissions**
   ```bash
   cd /var/www/phpfox
   chmod -R 755 ./
   chmod -R 777 ./file ./include/setting
   ```

3. **Create Database**
   ```sql
   CREATE DATABASE phpfox_db CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
   CREATE USER 'phpfox_user'@'localhost' IDENTIFIED BY 'your_password';
   GRANT ALL PRIVILEGES ON phpfox_db.* TO 'phpfox_user'@'localhost';
   FLUSH PRIVILEGES;
   ```

4. **Run Web Installer**
   - Navigate to `http://yourdomain.com/install`
   - Follow the on-screen instructions
   - Enter your database credentials
   - Create your admin account

### Method 2: Manual Installation

For advanced users who prefer manual installation:

1. Extract files to your web root
2. Configure database connection in `include/setting/server.sett.php`
3. Import database schema from `install/database.sql`
4. Set up initial configuration

## Post-Installation Steps

### 1. Secure Your Installation

After installation, it's important to secure your phpFox installation:

```bash
# Remove installation directory
rm -rf /var/www/phpfox/install

# Set proper file permissions
find . -type f -exec chmod 644 {} \;
find . -type d -exec chmod 755 {} \;
```

### 2. Configure Web Server

#### Apache Configuration

Create or edit `.htaccess` in your phpFox root:

```apache
<IfModule mod_rewrite.c>
    RewriteEngine On
    RewriteBase /
    RewriteCond %{REQUEST_FILENAME} !-f
    RewriteCond %{REQUEST_FILENAME} !-d
    RewriteRule ^(.*)$ index.php?do=/$1 [L,QSA]
</IfModule>
```

#### Nginx Configuration

Add to your server block:

```nginx
location / {
    try_files $uri $uri/ /index.php?$args;
}

location ~ \.php$ {
    fastcgi_pass unix:/var/run/php/php8.0-fpm.sock;
    fastcgi_index index.php;
    include fastcgi_params;
}
```

### 3. Configure Cron Jobs

Set up cron jobs for maintenance tasks:

```bash
# Edit crontab
crontab -e

# Add these lines
*/5 * * * * php /var/www/phpfox/cron.php
0 2 * * * php /var/www/phpfox/cleanup.php
```

## Troubleshooting

### Common Issues

**Issue**: White screen after installation
- **Solution**: Check PHP error logs, ensure all required extensions are installed

**Issue**: Database connection failed
- **Solution**: Verify database credentials, ensure MySQL is running

**Issue**: Permission denied errors
- **Solution**: Check file permissions, ensure web server has write access to required directories
