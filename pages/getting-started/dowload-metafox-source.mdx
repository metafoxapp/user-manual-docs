# Download MetaFox

Assume that you have purchased MetaFox license successfully. In this article, we will guide you to download the MetaFox source package.

Let's log into the [Client Area](https://clients.phpfox.com/) with your client account. 

Then go to the **Services** page. This page will list your active MetaFox license as the screenshot below

![Screenshot 2023-05-18 at 11.56.10.png](/images/Screenshot%202023-05-18%20at%2011.56.10.png)

## Download Installable Package
Click on an active MetaFox License in the **Product/Service** list to go to the Product Details page. 

In this page, you can click on the **Download Latest Version** button at the bottom of **License Information** section.

![Screenshot 2023-05-18 at 12.07.44.png](/images/Screenshot%202023-05-18%20at%2012.07.44.png)

## Download Frontend and Mobile Source Packages
If you want to download the source code for frontend and mobile app, you can click on the **Downloads** tab at the top of the Product Details page. 

In the **Downloads** section, you will see the list of source packages that you can download. You can click on the package version to download the source package that you want.

Note that Mobile Source (React Native) package are only available for Ultimate license.
