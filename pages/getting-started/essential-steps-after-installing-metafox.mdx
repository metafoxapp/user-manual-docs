# Essential steps after installing MetaFox

What should you do after installing MetaFox? This is a very common question asked by beginners. This article lists the most essential tasks you should perform immediately after installation.

1. Configure Site Name, Copyright, Meta Keyword, Meta Description — see [Configure Site info](/set-up-community/configure-general-and-site-settings/configure-site-info/)
2. Personalize phpFox site
   - Change your site's logo — see [Configure site logo, favicon and site assets](/set-up-community/configure-appearances-layout-and-themes/configure-site-logo-favicon-and-site-assets/)
   - Change your site's favicon (the tiny icon shown in browser tabs) — see [Configure site logo, favicon and site assets](/set-up-community/configure-appearances-layout-and-themes/configure-site-logo-favicon-and-site-assets/)
   - Change the visitor home page background image — see [Configure site logo, favicon and site assets](/set-up-community/configure-appearances-layout-and-themes/configure-site-logo-favicon-and-site-assets/)
   - Set default profile, pages and groups cover photos — see [Configure site logo, favicon and site assets](/set-up-community/configure-appearances-layout-and-themes/configure-site-logo-favicon-and-site-assets/)
3. Configure timezone and time format — see [Configure General and Site Settings](/set-up-community/configure-general-and-site-settings/)
4. Configure mail server — see [Set up SMTP Mail settings](/set-up-community/configure-general-and-site-settings/set-up-smtp-mail-settings/)
5. Configure spam protection with reCaptcha — see [Set up Google reCaptcha](/advanced-setups/set-up-google-recaptcha/)
6. Optimize PHP settings (for example: file upload limits, memory_limit) — see [Optimize PHP settings](/advanced-setups/optimize-php-settings/)
7. Configure maximum upload file size limits — see [Configure max file size by type](/set-up-community/set-up-storage/configure-max-file-size-by-type/)
8. Configure the content of privacy, terms and other static pages — see [Set up policy terms and other static pages](/set-up-community/configure-general-and-site-settings/set-up-policy-terms-and-other-static-pages/)
9. Set up the Contact Us form — see [Setup Contact Us form](/set-up-community/configure-general-and-site-settings/setup-contact-us-form/)
