# User Management

Effectively manage your community members with phpFox's comprehensive user management tools.

## User Overview

### User List

Access the user list at **AdminCP → Users → Manage Users**

View and manage all users with:
- Search and filtering
- Bulk actions
- Quick edit options
- User statistics

### User Search & Filters

Filter users by:

| Filter | Options |
|--------|---------|
| Status | Active, Inactive, Pending, Banned |
| Role | Admin, Moderator, Member, Custom Roles |
| Registration Date | Today, This Week, This Month, Custom Range |
| Activity | Active, Inactive (30+ days) |
| Email Verified | Yes, No |

## Adding Users

### Manual User Creation

Create users manually:

1. Navigate to **Users → Add New User**
2. Fill in required information:
   ```
   Username: john_doe
   Email: john@example.com
   Password: (auto-generate or custom)
   First Name: John
   Last Name: Doe
   ```
3. Select user role
4. Set initial status (Active/Pending)
5. Click "Create User"

### Bulk User Import

Import multiple users:

1. **Users → Import Users**
2. Download CSV template
3. Fill in user data:
   ```csv
   username,email,first_name,last_name,password
   john_doe,john@example.com,John,Doe,Pass123!
   jane_smith,jane@example.com,Jane,Smith,Pass456!
   ```
4. Upload CSV file
5. Map columns
6. Start import

### Custom Profile Fields

Add custom fields:

1. **Users → Profile Fields**
2. Click "Add New Field"
3. Configure field:
   ```javascript
   {
     "fieldName": "occupation",
     "fieldType": "text",
     "required": false,
     "showOnProfile": true,
     "searchable": true
   }
   ```

Available field types:
- Text input
- Textarea
- Dropdown select
- Multiple choice
- Date picker
- File upload

## User Roles & Permissions

### Default Roles

phpFox comes with these roles:

#### Administrator
- Full system access
- Manage all content
- Configure settings
- Manage users and roles

#### Moderator
- Manage content
- Moderate users
- Handle reports
- Limited settings access

#### Member (Regular User)
- Create content
- Interact with others
- Join groups
- Standard features

### Creating Custom Roles

Create specialized roles:

1. **Users → User Roles → Add New**
2. Name your role: "VIP Member"
3. Configure permissions:

```javascript
{
  "canPost": true,
  "canComment": true,
  "canUploadVideos": true,
  "maxPhotoSize": 20971520, // 20MB
  "canCreateGroups": true,
  "canCreateEvents": true,
  "adsDisabled": true,
  "specialBadge": "VIP"
}
```

## User Status Management

### User Status Types

**Active**
- Full access to site
- Can login and participate
- All features enabled

**Inactive**
- Account exists but disabled
- Cannot login
- Profile hidden from search

**Pending**
- Awaiting email verification or admin approval
- Limited access
- Cannot fully participate

**Banned**
- Access denied
- Cannot login
- Profile hidden

### Activating/Deactivating Users

Change user status:

1. Find user in user list
2. Click user's name
3. Change status dropdown
4. Select new status
5. Add reason (for bans)
6. Save changes

### Batch Actions

Perform bulk operations:

```
Select users → Choose action:
├── Activate
├── Deactivate
├── Delete
├── Change Role
├── Send Email
└── Export Data
```

## User Moderation

### Warning System

Issue warnings to users:

1. **User Profile → Moderation**
2. Click "Issue Warning"
3. Select warning type:
   - Spam
   - Inappropriate content
   - Harassment
   - Terms violation
4. Add notes
5. Set expiry (optional)
6. Notify user

Warning thresholds:
```
1-2 warnings: Notice sent
3 warnings: 24-hour suspension
4+ warnings: Permanent ban (review required)
```

### Banning Users

Ban problematic users:

#### Temporary Ban
```javascript
{
  "userId": 12345,
  "duration": "7 days",
  "reason": "Spam posting",
  "notifyUser": true
}
```

#### Permanent Ban
```javascript
{
  "userId": 12345,
  "permanent": true,
  "reason": "Severe ToS violation",
  "banIP": true,
  "banEmail": true
}
```

### IP Banning

Block by IP address:

1. **Users → Ban Management → IP Bans**
2. Add IP address or range
3. Set duration
4. Add reason
5. Save

Examples:
```
Single IP: 192.168.1.100
IP Range: 192.168.1.0/24
Multiple: 192.168.1.100, 192.168.1.200
```

## Privacy & GDPR Compliance

### Data Export

Users can export their data:

1. User settings → Privacy → Export Data
2. System generates archive
3. Download includes:
   - Profile information
   - Posts and comments
   - Photos and videos
   - Messages
   - Activity history

### Account Deletion

Handle deletion requests:

**Soft Delete** (Recommended)
- Mark account as deleted
- Hide from public
- Retain data for period
- Can be restored

**Hard Delete**
- Permanently remove data
- Cannot be undone
- GDPR compliant
- Anonymize content

## Troubleshooting

### Common Issues

**Users can't register**
- Check registration settings
- Verify email configuration
- Test captcha functionality

**Users locked out**
- Check failed login attempts
- Reset password
- Verify account status

**Spam registrations**
- Enable stricter captcha
- Require email verification
- Enable admin approval
- Block disposable emails
