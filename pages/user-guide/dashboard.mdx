# Dashboard Overview

The phpFox admin dashboard is your central hub for managing your social network. This guide will help you understand and navigate the dashboard effectively.

## Dashboard Layout

### Top Navigation Bar

The top navigation bar provides quick access to:

- **Site Preview** - View your site as a visitor
- **Notifications** - Admin notifications and alerts
- **Quick Actions** - Common administrative tasks
- **Profile** - Your admin profile and settings
- **Logout** - Sign out of admin panel

## Dashboard Customization

### Widget Management

Customize your dashboard by:

1. **Adding Widgets**
   - Click "Add Widget" button
   - Select from available widgets
   - Configure widget settings

2. **Rearranging Widgets**
   - Drag and drop widgets
   - Resize widget containers
   - Create custom layouts

3. **Removing Widgets**
   - Click the "Ã—" on any widget
   - Confirm removal
   - Restore from widget library

4. Download report

## Troubleshooting

### Dashboard Not Loading

If dashboard doesn't load:

1. Clear browser cache
2. Check server status
3. Verify database connection
4. Review error logs

### Slow Dashboard Performance

To improve performance:

1. Reduce number of widgets
2. Enable caching
3. Optimize database
4. Upgrade server resources
