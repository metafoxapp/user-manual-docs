# API Endpoints

Complete reference of available API endpoints.

## Users

### Get Users

```http
GET /api/v1/users
```

Response:
```json
{
  "status": "success",
  "data": [
    {
      "id": 1,
      "username": "john_doe",
      "email": "john@example.com",
      "created_at": "2026-01-01T10:00:00Z"
    }
  ]
}
```

### Get User by ID

```http
GET /api/v1/users/{id}
```

### Create User

```http
POST /api/v1/users
Content-Type: application/json

{
  "username": "new_user",
  "email": "user@example.com",
  "password": "secure_password"
}
```

### Update User

```http
PUT /api/v1/users/{id}
Content-Type: application/json

{
  "email": "newemail@example.com"
}
```

### Delete User

```http
DELETE /api/v1/users/{id}
```

## Posts

### Get Posts

```http
GET /api/v1/posts
```

### Create Post

```http
POST /api/v1/posts
Content-Type: application/json
```

### Update Post

```http
PUT /api/v1/posts/{id}
```

### Delete Post

```http
DELETE /api/v1/posts/{id}
```

## Photos

### Upload Photo

```http
POST /api/v1/photos
Content-Type: multipart/form-data

file: [binary data]
caption: "My photo"
```

### Get Photos

```http
GET /api/v1/photos
```

## Comments

### Get Comments

```http
GET /api/v1/posts/{post_id}/comments
```

### Create Comment

```http
POST /api/v1/posts/{post_id}/comments
Content-Type: application/json

{
  "text": "Great post!"
}
```

## Query Parameters

Common query parameters:

| Parameter | Type | Description |
|-----------|------|-------------|
| page | integer | Page number (default: 1) |
| per_page | integer | Items per page (default: 20, max: 100) |
| sort | string | Sort field |
| order | string | Sort order (asc/desc) |
| search | string | Search query |

Example:
```http
GET /api/v1/users?page=2&per_page=50&sort=created_at&order=desc
```

## Webhooks

Subscribe to events:

```http
POST /api/v1/webhooks
Content-Type: application/json

{
  "url": "https://your-app.com/webhook",
  "events": ["user.created", "post.created"]
}
```
