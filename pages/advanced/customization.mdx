# Customization

Learn how to customize phpFox to match your brand and requirements.

## Theme Customization

### Visual Customization

Customize your site's appearance through the theme customizer:

1. **AdminCP → Appearance → Theme Customizer**
2. Adjust settings:
   - Colors and fonts
   - Layout options
   - Logo and branding
   - Custom CSS

### Color Schemes

Define your brand colors using CSS variables for consistent theming across your site.

### Custom CSS

Add custom styles to personalize your site's look and feel.
