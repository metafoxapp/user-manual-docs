# Plugins & Extensions

Extend phpFox functionality with plugins and extensions.

## Plugin Marketplace

Browse and install plugins from the phpFox marketplace.

## Installing Plugins

1. Download plugin
2. Upload to /plugins directory
3. Activate in AdminCP
4. Configure settings

## Creating Custom Plugins

Develop your own plugins to add custom functionality.

## Popular Plugins

- Social login integrations
- Payment gateways
- Advanced analytics
- SEO tools
