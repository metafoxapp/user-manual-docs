# MetaFox 5.2.4

**Release Date: Jun 17th, 2026**

## What's new

### **Core**

* **Octane Compatibility** – fully compatible with Laravel Octane, enabling high-performance, long-lived application serving.
* **AdminCP – Inactive Members Filter** – the logic for identifying inactive members has been updated, giving administrators a more accurate picture of user engagement.
* **Issues/Improvements**
    * Frontend Web
        * Added a new setting **Search Method** in global search that allows admins to choose between `Keyword Search` and `Partial Match Search`.
        * Fixed an issue where users could not edit the position of their cover photo.
        * Fixed an issue where the background color on the Register Page could not be edited.
        * Fixed a bug that allowed users to view comments from blocked users.
    * Mobile App
        * The app now displays the latest data immediately upon opening.
        * Fixed a missing feed info issue when a user checks in to a location.
        * Fixed an issue where the main menu color was reset after a site upgrade.
        * Fixed a UI issue when adding a new comment.

### **ChatPlus**

* Fixed a missing incoming call sound on mobile browsers.

### **Invitations**

* Fixed an issue where the invitation status was not updated after the invitee completed sign-up.

### **And,**

* There are some additional bug fixes, minor UI/UX improvements, and stability updates across other apps and components in MetaFox 5.2.4.
