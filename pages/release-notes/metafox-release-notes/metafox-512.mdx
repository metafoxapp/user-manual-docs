# MetaFox 512

Release Date: Jun 14, 2023

### What's new

-   New Apps / Features
    -   Invitations 5.0.0
    -   Data Migration from phpFox V4
-   Improvements:  
    -   Able to use different domains for Web (Frontend), API (Backend) and AdminCP
    -   Update workflow when buying an item with Activity Points
    -   Supports filtering of featured items
    -   Add breadcrumbs to Create/Edit forms of items if their owner is not a user (ex: Page, Group, ...)
    -   Improve quality audio/video call  
        
    -   Improve performance
-   Bug fixes
