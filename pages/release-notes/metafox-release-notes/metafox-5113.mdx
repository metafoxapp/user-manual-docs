# MetaFox 5113

**Release Date**: Sep 10th, 2024

### Core 

-   Mobile apps 
    -   Upgrade Facebook SDK
-   UI/UX Improvements
    -   Frontend Web  
        -   Localization - Support detecting language based on the browser setting.
        -   Login with MFA
            -   User will be back to the original page after logging in using MFA.
            -   First authentication option should be automatically selected by default.
        -   Unify the icon style across the platform using the outlined variant.
        -   Sign-In - Improve the layout of the Social Login button
        -   Scroll to the top of the Homepage when the user is on the Homepage and clicks on Home menu or Site logo
        -   Authenticator page - Hide Create account link
    -   AdminCP
        -   Localization - Countries:
            -   Make Fips Code, Geonames Code optional when creating/editing states.
            -   Add action to add new items to the Options menu.
        -   Members/Inactive: Support auto-compete on City field.
        -   SEO
            -   Improve Apps filter in SEO settings manage page
            -   Improve SEO manage settings page
        -   Schedule Jobs - Add the Settings page with detailed cron commands
        -   Point Settings - Users can input the integer starting with 0 in the Point Settings
        -   Mobile Services/Smart Banner - Moving Google/Apple ID Settings from Admob page to the Smart Banner page
        -   Email Inactive Members - Improve the UI/UX for processing inactive email.
        -   Health Check: improve the process of checking the queue worker state.
        -   Install / Reinstall App - Display popup whenever the installation process is successful
        -   Login with MFA - Admin will be redirected to AdminCP via the AdminCP Login form
        -   Storage: hide storage configurations unless they are properly configured.
    -   Mobile Apps
        -   Remove the redundant line on the profile page (Android).
        -   Improve the Featured/Sponsored label on the feed item.
        -   Settings: Improve the layout for the action sheet header when viewing some setting pages
        -   Improve phrase for the Privacy menu

-   Support Progressive Web App on Chrome
-   Support settings to require users in specific roles to enable MFA.
-   SEO - Support descriptive URLs.
-   Toggle Site: support WYSIWYG editor for Offline Message
-   Site Statistic: Add a new page to manage Site Statistics.
-   Notification: Allow administrators to disable notifications for the entire app.
-   Brute-force Protection: improve the brute-force workflow.
-   Performance Improvement - Not to reset the global cache when performing specific actions
-   AdminCP Site Settings - Support translating some components on the Site Settings page.
-   Email Inactive Members: Process inactive email in batch to avoid being marked as spam
-   Flatten Theme - Move background default to asset
-   Support environment variable to control site backups.
-   Installation Wizard: Improve validation for database version
-   Smart Banner: Support setting to allow administrators to reposition the Smart Banner.
-   Fix bugs
