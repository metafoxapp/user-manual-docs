# MetaFox 520

### What's new

**Core**

-   Upgrade to Laravel 12 - Enhanced performance and security with the latest Laravel framework
    
-   Compatible with PHP 8.4 - Support for the newest PHP version ensuring optimal performance and security
    
-   Compatible with PostgreSQL 17 - Extended database compatibility for enterprise deployments
    
-   React Native 0.77 Support - Mobile apps now support Android 15 with the latest React Native framework
    
-   API Versioning Support - Better API management and backward compatibility for developers
    
-   Image Quality Control - New setting that allows users to toggle between viewing original uploaded images and resampled versions for better performance
    
-   Contact Form Enhancement - Added Terms of Service confirmation checkbox to contact forms for better compliance
    
-   CSV User Import - Administrators can now import users from CSV files with sample file download support
    

**AdminCP Improvements**

-   Message Queue Timeout Setting - Configure queue timeout settings for better performance management
    
-   Password Reset Link Expiration - Configurable expiration time for password reset links
    
-   Member Management Enhancements:
    

-   Filter members by Banned User role
    
-   Improved workflow for reassigning super administrators
    
-   Enhanced custom field handling with better error messages
    

-   Upgrade Wizard Improvements - Better handling of incompatible apps during upgrade process
    
-   Performance Optimization - Faster language package setup during upgrades
    

**UI/UX Improvements**

-   Registration Enhancement - Users are automatically logged in after completing email verification
    
-   Social Login Integration - Added signup buttons for social accounts on the registration form
    
-   SEO Improvements - Customized SEO metadata now takes precedence over system-generated content
    
-   Enhanced Search Functionality - Search results now include descriptions across all apps
    
-   Dark Mode Improvements - Better layout and color consistency in dark mode
    
-   Mobile Layout Enhancements - Improved video layout for directly uploaded content
    

**Photos**

-   Album Management - Enhanced photo album creation and editing experience
    
-   Search Improvements - Better photo search functionality, including description-based searches
    

**Social Connect**

-   Enhanced Social Login - Improved Facebook and Google login experience
    
-   Avatar Handling - Better avatar resolution when logging in with social accounts
    
-   Mobile Stability - Fixed crashes when logging in with social accounts on Android
    

**Comments**

-   GIF Support Enhancement - Improved description handling for comments containing Giphy content
    
-   Edit History - Enhanced edit history descriptions for better content tracking
    

**Events**

-   UI Improvements - Better event title display in Upcoming Events blocks
    
-   Group Integration - Enhanced event invitation system for group members
    

**Groups**

-   Menu Settings - Ability to deactivate default menus for better customization
    
-   Landing Page Control - Improved UX for menus set as landing pages
    
-   Role Management - Better role label display in dark mode
    

**Pages**

-   Menu Customization - Enhanced menu settings with the ability to deactivate default options
    

**Music**

-   Mobile Enhancements - Fixed repeat/shuffle functionality on mobile layouts
    
-   Playlist Management - Improved album and playlist editing experience
    

**Videos**

-   Mobile Layout - Enhanced video display layout for mobile applications
    

**Subscriptions**

-   Translation Support - Multi-language support for subscription packages
    
-   Role Management - Improved user role handling during subscription changes
