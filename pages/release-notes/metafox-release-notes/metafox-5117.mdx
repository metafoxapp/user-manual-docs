# MetaFox 5117

**Release Date**: Feb 12th, 2025

### Core

-   Notification
    -   Add ability to mark individual notification as \[Read\] or \[Unread\] instead of all of them
-   AdminCP
    -   Support Manage Items page.
    -   Support Mass Actions to admin can delete/ approve all items
    -   Site Settings - PWA: The App Description and Install Description can be translatable.
    -   Deny User: Administrators can either send an email or SMS when denying a user account.
-   Support CLI commands to activate/deactivate an app.
-   Check & refactor the URLs to avoid being matched against standard firewall rules.
-   UI/UX Improvements
    -   AdminCP
        -   Relationship - Remove the redundant setting "Enable Relationships".
        -   App - Installed
            -   Support sortable columns
            -   Add the Upgrade button to upgrade all apps
            -   Support mass actions to activate/deactivate apps.
        -   Members - Update phrases on the Search form of Inactive Members 
            -   Change the Search field to Display Name field.
            -   Change the "Any" role text to "All".
        -   Show the total items on all data grid pages.
    -   Mobile Apps
        -   Improve the pointer's behavior on screens having a text field/ area when closing the keyboard.
        -   User Profile: add options to Approve or Deny a user account on the user profile page.
    -   Frontend Web:
        -   Improve Shortcut dialog popup.
        -   Remove the horizontal bar on the notification dialog.
        -   CK Editor - Remove the action sheet when users input the link to the content field.
        -   Improve the filter field on the transaction/invoice page.
        -   Improve the Layout when users select the location.
-   Fix bugs
