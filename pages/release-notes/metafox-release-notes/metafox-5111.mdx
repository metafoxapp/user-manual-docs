# MetaFox 5111

**Release Date: June 26th, 2024**

### Core 

-   Admin can define custom error messages for different contexts.
-   UI/UX Improvements on Web
    -   Layout for Text Displaying on Presets of Editor Mode, Navigation.
    -   Pagination on the Members page
    -   Layout when viewing the feedback on the Cancel Members page.
    -   Layout of Manage notification setting section when editing user
    -   Unify the UX of the 'Add New' menu actions in AdminCP.
    -   Description of the "Verify After Changing Phone Number" setting.
    -   Description of some Quota Control's settings.
    -   Manage Reports - Hide the search form when there are no reported items.
    -   Manage Members: Rephrase "Pending Verification Email / Phone Number" as "Unverified Email / Phone Number"
    -   Users can input a number to modify the photo scale on CK Editor.
    -   Login Popup - Add a link to the registration page and social login buttons.
    -   Keep the current page(URL) when logging in to AdminCP instead of redirecting to the Dashboard
-   AdminCP
    -   Manage Members:
        -   Able to filter members by registration date and country.
        -   Display the total count of search results.
        -   Update the URL on the address bar with search parameters when searching.
        -   Hide the Ban User option on the Super Admin.
    -   Notify users having administrative privileges to approve members when new pending members are awaiting approval.
    -   User Settings: revise the "User Role Filter Excludes" setting to control whether users of a specific role can be searched.
    -   Custom Fields - The admin can reorder the Basic Information section
-   Improve the query performance when checking for blocked user relations.
-   Add Payment Settings group menu in Account Settings
-   Improve Payment app for third-party gateways
    -   Support rules and request validation for 3party gateways in all payment apps
    -   Support multiple-step payment form.
-   Can search Members by applicable custom fields upon choosing a specific role.
-   Show item's category on the Item Detail page even though the category has been disabled.
-   Support opt-in for using email and SMS services on the Registration form
    -   Add a new setting to control whether the opt-in agreement should be shown in the registration process.
    -   Users will still receive email/SMS for Email Welcome, Verification Code , Authentication Code even though they don't check this opt-in during Registration.
-   Remove default MetaFox Favicon for Fresh Site Installation
-   Account Setting - Notifications
    -   Add a setting that allows users to control the entire notification channel.
    -   Add the SMS Notifications page
-   Mobile Apps
    -   The admin can configure menu items of the Account Settings menu in AdminCP.
    -   Use logo configured in AdminCP on the Login screen.
    -   Show No Internet page when device is offline
    -   Add the option "Copy Link Profile" on the User Profile
    -   Support grid layout for app menu and account settings menu
    -   Users can unselect the current State/City on the Edit Profile Info page.
    -   Add the search field when viewing the gender (I am) drop-down.
-   Fix bugs
