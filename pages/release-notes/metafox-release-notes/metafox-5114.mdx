# MetaFox 5114

**Release Date**: Nov 10th, 2024

### Core 

-   SEO: Admin can control bot indexing and configure canonical URL for specific pages
-   Improve performance on Frontend web 
-   GDPR Cookie Privacy: Administrators can change the URL to the cookie policy page.
-   Custom Field - Support multiple languages for Multiple Select option
-   SEO: Support descriptive URLs in all apps
-   Regex Rules - Apply the Setting "Display Name Regex Rule" for Group/Page
-   Support config Light/Dark mode for As Guest user
-   Currencies - Support administrators to list users who are using a specific currency.
-   Notification
    -   Admin can check whether a notification type channel can be enabled/disabled based on the type configurations
    -   Suppress notifications from being sent to banned users.
    -   Suppress emails / SMS messages from being sent to unverified emails / phone numbers.
-   UI/UX Improvements
    -   Frontend Web  
        -   Improve slider form field
        -   On the Search Result,  improve layout for click icon when hovering on the Reset link.
        -   When dragging a photo to reorder photo items on the Create/Edit page, the photo of this position must be pushed to the next position.
        -   When users open the Check-in modal / Location field on Composer post/ Marketplace/ Event, the suggested location list will be displayed based on the current location.
        -   Improve the warning message for Multi-Factor Authentication, SignIn, Registration Settings, etc.
        -   MFA - Users should be navigated directly to the authentication code step if only one authentication method is available.
        -   Add the Follow/Unfollow actions on the "Profile Popup Viewer"
        -   Support spell check on Chrome default (Feed Post Editor )
        -   Hide Uploading Image file on Text Editor for Guest Users
        -   Check-In - Improve the layout on the scroll bar on Dark Mode.
    -   AdminCP
        -   Members
            -   Hide the city drop when no country is selected.
            -   The city dropdown now supports automatic completion.
            -   Fix the expanded field after searching. They only collapse when clicking on the "View less" link.
            -   Update success message when admin approve the user
            -   Change the ordering on the Cancelled Members
        -   Dashboard - Add Pending Reports metric under Site Statistics.
        -   Report Items: the 'Process Report' action is  relabeled as 'Mark as Processed'
    -   Mobile Apps
        -   After changing password and logging out, users are required to log in again when tappping on the saved profile.
        -   Support Deep links with multiple domains. 
        -   Improve the layout of  Mutual Friends pop-up
-   Mobile apps 
    -   Support iPhone 16 device
-   Fix bugs
