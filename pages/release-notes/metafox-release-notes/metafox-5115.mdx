# MetaFox 5115

**Release Date**: Dec 2nd, 2024

### New Feature - Strengthened Security with Change File Checksum Verification

  
Checksum verification ensures the integrity and authenticity of your MetaFox files. With this feature, administrators can:  
    •    Detect unauthorized changes to core files, configurations, or scripts.  
    •    Maintain a secure and consistent deployment environment.  
    •    Quickly identify modifications that could compromise security or lead to functionality issues.

![Screenshot 2024-12-02 at 11.44.39.png](/images/Screenshot%202024-12-02%20at%2011.44.39.png)

  
This enhancement is particularly valuable in guarding against potential threats, such as malware injections or unauthorized updates, giving you peace of mind that your MetaFox installation remains secure and reliable.
