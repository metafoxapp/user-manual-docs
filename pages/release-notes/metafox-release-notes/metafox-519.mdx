# MetaFox 519

**Release Date: Apr 12, 2024**

## **Core Updates**

### UI/UX Improvements

-   Enhance the typing experience with the new Rich Text Editor.
-   Hide the placeholder when users fill in saved data in the Username field on the Login page.
-   Enhance the profile popup position when hovering over the User/Page/Group profile.
-   Enhance the visibility of the Feedback column in the Canceled Members page in AdminCP.
-   Improve the color consistency of the pagination controls on some pages (Point Settings, Phrases).
-   Support general player button control for video
-   Improve the Turn Off Two-factor Authentication popup.
-   Improve the Display Name column when searching Members in AdminCP.
-   Enhance the layout of the Registration form with less than 5 fields.
-   Enhance wording on the Global Search box.
-   Improve the comment box on the feed item on mobile
-   In User Profile, scroll back to the top of the profile page instead of opening the user story page when clicking on a sticky avatar icon.

### Localization

-   Show the original translation key when enabling the Active Translation setting.
-   Replace the "User Group" with the "Role" in phrases.

### Mobile apps

-   Display the Previous Sign-In screen instead of the Sign-In screen if the user has previously logged in.
-   Update React Native Firebase library to the latest.
-   Support the "Profile Menu Settings" page

### AdminCP

-   Members - Show confirm popup when clicking on the Logout All Users button
-   Appearances - Themes: Support modifying variant thumbnails.
-   Appearances - Should not reload the page after editing the theme variant
-   Custom Fields - Use the toggle button instead of text (Yes/No) at Active column
-   Custom Fields -  Optimize the column width in the Manage Custom Fields page**.**
-   Add a warning to remind for Migration from legacy FCM APIs
-   Edit text color: Automatically change the color code when the admin points to another place in the Color Picker

### Others

-   Layout Editor - Only display newly added block within the layout editor after saving.
-   Firebase - Migrate from legacy FCM APIs to HTTP v1
-   Forgot Password: Enhance the security of the workflow by limiting the maximum number of attempts to get the verification code.
-   Custom Fields - Enhance the workflow of role-specific fields by supporting multiple roles and displaying them on registration accordingly.
-   Email - Support predefined variables to enhance the customizability of the email greetings.
-   Account Setting - Hide settings of inactive Apps in Account Settings > Privacy Settings > Your Profile Settings & Sharing Items
-   Pagination - Support "Go To Page" option so users can quickly navigate through paginated items.
-   Rich-text Editor - Support adding link to an image on Rich-text-editor
-   Notification - Support deleting all notifications on the frontend web
