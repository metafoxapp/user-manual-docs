# MetaFox 521

Release Date: Dec 23rd, 2025

## **What's new**

### **Core**

-   Sponsorship prompts now support item\_name parameter for better conversions.
-   AdminCP: more explicit Service Keys guidance to reduce setup confusion.
-   Mail: unsubscribe link added to email footer for compliance.
-   Statistics chart selections now stay cached for faster admin reviews.

### **ChatPlus**

-   Message migration from Chat to ChatPlus.
-   Upgraded Rocket.Chat engine to 7.x for stability and performance.
-   Mobile support updated for iPhone 17/17 Pro/17 Pro Max.
-   Web: shows message limit warnings before sending to avoid loss of content.

### **Social Registration**

-   Added opt-in step for Terms of Service and notifications before account creation, improving consent flow.

### **Mobile experience**

-   Web/Mobile push notification setting screens.
-   Support Google Play's 16 KB page sizes.

### **Media & sharing**

-   Fixed TikTok video sharing errors.

### **Reliability highlights**

-   Fixed Sort Feed on mobile app.
-   Fixed group invitation links now open with the correct URL and pending requests stay pending until reviewed .
-   Localization fixes for deactivating unused countries and currencies in AdminCP.
-   Other minor fixes and improvements.
