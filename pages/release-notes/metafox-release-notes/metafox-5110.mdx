# MetaFox 5110

**Release Date: May 22, 2024**

## **Core  and App Updates**

### Featured Items (NEW)

#### User Features

1.  **Dashboard**
    
    -   Overview of featured items
    -   Quick access to invoice management
2.  **Item Management**
    
    -   Submit items to be featured
    -   Track status of submitted items
3.  **Invoice Management**
    
    -   View and download invoices
    -   Payment history
    -   Outstanding payments
    -   Payment methods (credit card, PayPal, etc.)
4.  **Package Selection**
    
    -   Browse and select packages for featuring items
    -   Detailed package descriptions and pricing
5.  **Notifications**
    
    -   Email and in-app notifications for invoice due dates

#### Admin Features

1.  **Dashboard**
    
    -   Overview of total earnings
    -   Quick access to manage invoices, packages, and transactions
    -   System alerts and notifications
2.  **Invoice Management**
    
    -   View and manage all user invoices
    -   Create, edit, and delete invoices
    -   Track payment statuses
    -   Generate reports on invoice activities
3.  **Package Management**
    
    -   Create, edit, and delete featured packages
    -   Set pricing, duration for each featured package
    -   Special promotions and discounts management
4.  **Transaction Management**
    
    -   View and manage all transactions
    -   Track payment gateway integrations
    -   Refund processing
    -   Transaction history and reporting
