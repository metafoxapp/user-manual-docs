# MetaFox 518

**Release Date: Mar 5, 2024**

### **Core Updates**

-   Support the new Rich-text Editor

-   Enable Rich-text Editor for Profile - About Me

-   Localization

-   Countries - Support manage cities
-   Phrases - Replace "Group" column with the "Original Text" column.

-   Layout Editor Mode:

-   Able to add Google Adsense block
-   Support Editor Mode on the Registration page
-   Update link for the Help icon

-   Minor UI/UX Improvements:

-   Display the loading screen based on chosen light/dark mode.
-   Support display menu icons of static pages on mobile apps
-   Add the Close button at the right of the search field when users fill data to this field on the State/City screen
-   Improve phrases of Reports
-   Improve the button color of Sign in with Google or TikTok if it is the unique option
-   Mobile apps - Display calender picker when tapping on the calendar icon
-   Extend the forgot password session lifetime to 5 minutes and improve the error message to avoid confusion

-   Users:

-   Admin can log out all users
-   User can log out his/her account on other devices
-   Remove the custom gender to avoid discrimination
-   Registration Settings: Improve the workflow of the "Force Users to Upload Profile Image" setting
-   Disable the Registration option for custom fields of specific roles

-   Support parse email addresses in the text content
-   Appearances

-   Can configure icon color of side menu items of the mobile apps.
-   Can open link-type menus with context menu (right-click menu)

-   Fix bugs

  

### **App Updates**

-   All apps

-   Improve phrases
-   Manage Categories - Can view list of subcategories by clicking on the number of subcategories

-   Events - Include group events when listing events that users are interested in, attending or invited to.
-   Follow - Add the 'Follow User' option on the Following/Follower sub-tab on the profile page
-   Photos

-   Support edit photo after selecting a photo
-   Add permission settings to allow users to add mature images with warnings
-   Album Details - Not reload when deleting photo of the album

-   Videos

-   Improve video resolution
-   Reposition the Fullscreen option on iPad
-   Able to share Rumble Video Platform

-   Pages

-   Support custom fields
-   Enable the Like button on the shared-page feed items

-   Newsletters - Admin can select multiple roles, genders and locations for a newletter
-   Chat - Support push notifications on Web/Mobile app when users receive a new message
-   ChatPlus

-   Add the "Jump to this message" action on the pinned messages
-   Can jump to the original message when clicking on the reply message or searched message
-   Users can copy/paste and drag/drop multiple files (photo/video/...) into the conversation

-   Stories

-   Users can mute/unmute on the Video Story on web
-   Add friend suggestions for the Instagram-type story

-   Advertise - Improve the layout in the Manage Ads page
-   Comment - Add the "View Post" button on the Comment screen of mobile apps
-   Feed

-   Add the permission setting "Can save media posts?" in the ACP
-   Change permission setting "Can pin activity feed?" to "Can pin posts on Profile User/Page/Group?"

-   Invite - Admin can manage invitations of all users in AdminCP
-   Reactions - Admin can manage existing reactions in AdminCP
-   EWallet - Improve layout on mobile apps
-   Hashtags - Support multiple languages
-   Activity Points - Add permission setting to allow users to create point conversion requests
-   Status Background - Support multiple language for the Collection title
-   And, many bug fixes of apps
