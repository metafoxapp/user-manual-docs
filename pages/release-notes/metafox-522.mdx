# MetaFox 5.2.2

**Release Date: Mar 11th, 2026**

## **What's new**

### **Core**

* Improved Installation UI - A refreshed and more intuitive installation interface for a smoother setup experience.
* Age Verification on Signup - Administrators can now enable age verification during the registration process to ensure compliance with age-related policies.
* Dark Mode Enhancements - Updated scroll bar colors in Dark Mode for better visual consistency.
* Guest Display Mode - Fixed the "Enforce Display Mode" setting to correctly apply to guest users.
* Changing notification channels in AdminCP now works correctly for all notification types.
* Health Check - the "Reachable URLs" check no longer reports false failures.
* The Terms of Use page is now accessible from the registration page.

### **Feed**

* The "View More" link now correctly appears when post text is too long.
* Users can now share Facebook and Rumble videos without errors.
* Fixed timeline timestamps on activity posts to match the website display.

### **Music**

* Fixed the "Music Max Upload Size (bytes)" setting not being applied correctly.
* Users can now add songs from a playlist detail page even when they do not have permission to edit the playlist.

### **Videos**

* URL-based videos are now displayed correctly.
* Fixed an issue on Android where the comment bar overlapped the phone's navigation buttons.

### **Stories**

* Fixed a crash that occurred when users reacted to stories.
* Fixed screen flickering when the app transitions to the next story.
* Fixed the Edit Video feature for stories on Android devices.
* Improved the layout when users crop a video story on Android.

### **Events**

* Users can now export and add upcoming events to their calendar correctly.

### **ChatGPT BOT**

* Fixed an issue preventing users from sending messages to the chatbot.

### **Social Connect**

* An appropriate error message is now shown when users cancel social login.
* Fixed an issue where iOS users were unable to log in with Facebook.

### **Saved Items**

* Fixed incorrect user attribution when an item is added to a collection.

### **Forums**

* Users can no longer create threads in forums where they do not have the required permission.

### **Live Videos**

* Fixed an issue preventing iOS users from starting a live video.

### **ChatPlus**

* Compatible with MetaFox 5.2.2
* Fix bugs

### **And,**

* There are other minor fixes, UI/UX improvements, and compatibility updates across other apps.
