Set up Apple Pay

This guide aims to assist you in seamlessly integrating Apple Pay as a payment method on your MetaFox platform. With Apple Pay, your users can make secure transactions using their Apple devices, enhancing their payment experience.

Assume that you have installed the Apple Pay app on your MetaFox successfully. You may want to refer the for app if needed.

Lets go to AdminCP > App Settings > Payment. You will see the Apple Pay item listed as below

Edit the Apple Pay and choose a payment processor you want to use with Apple Pay. You will need to make sure that the chosen Payment Processor has been set up properly. You can refer to the guide for more detail.

Then, Check the Active checkbox and save changes to complete the set up for Apple Pay.

Congratulations. You have done the setup for Apple Pay on MetaFox site!
