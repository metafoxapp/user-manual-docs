Release NotesMetaFox - Release NotesMetaFox 5.1.10

MetaFox 5110

Release Date: May 22, 2024

Core and App Updates

Featured Items (NEW)

User Features

  1. Dashboard

    • Overview of featured items
    • Quick access to invoice management

  2. Item Management

    • Submit items to be featured
    • Track status of submitted items

  3. Invoice Management

    • View and download invoices
    • Payment history
    • Outstanding payments
    • Payment methods (credit card, PayPal, etc.)

  4. Package Selection

    • Browse and select packages for featuring items
    • Detailed package descriptions and pricing

  5. Notifications

    • Email and in-app notifications for invoice due dates

Admin Features

  1. Dashboard

    • Overview of total earnings
    • Quick access to manage invoices, packages, and transactions
    • System alerts and notifications

  2. Invoice Management

    • View and manage all user invoices
    • Create, edit, and delete invoices
    • Track payment statuses
    • Generate reports on invoice activities

  3. Package Management

    • Create, edit, and delete featured packages
    • Set pricing, duration for each featured package
    • Special promotions and discounts management

  4. Transaction Management

    • View and manage all transactions
    • Track payment gateway integrations
    • Refund processing
    • Transaction history and reporting

Core

  • UI/UX Improvements

    • On User Profile when switching tabs or showing user details
    • When editing items that belong to deactivated categories

  • Custom Fields

  • Enable validation on the Text field with regex rules.

  • Support sortable options for Multiple Selection field

  • Able to control role-based field visibilities.

  • AdminCP - Storage: Add setting to control file deletion upon removing related entities.

Feed

  • Support translation on posts with Google Translate
  • Administrators can delete feeds and all of its related items.
  • Support new permission to control How many posts per day that user can post?
  • Support Setting to control maximum characters for Post Status
  • UI/UX Improvements
    • The layout on the Feed Types table
    • Change the Title on the Edit Post screen on mobile apps
    • Space between the keyboard and icon bar when sharing the feed

Pages

  • Integrate with the Featured Items app
  • Users can input an External Link when creating a page.
  • UI/UX Improvements
    • AdminCP - Manage Custom/Section fields: Add Add Custom Field/ Add Section link to the manage page.
    • Change the label on the Manage Group screen on mobile app.
    • Hide profile/cover photoon page detail after the admin deletes the photo.
  • Users can upload a profile picture for the page when creating it.

Groups

  • Integrate with the Featured Items app
  • Manage Group:
    • Allow group administrators to view the history of Membership Requests.
    • Support Membership Requests on mobile apps
  • Allow non-registered users to access secret groups via invitation links.
  • UI/UX Improvements
    • On workflow of sharing an item link from a Closed/Secret group to the discussion in the same group.
    • Add a confirmation message before saving the group privacy setting.
    • The success message when users invite friends to join the group.
    • Change the label on the Manage Group screen on mobile app.
    • AdminCP -Manage Custom/Section fields: Add Add Custom Field/ Add Section link to the manage page
  • Add Who can view members? permission to control who can view the group members.

Comments

  • Support translation on comments/replies with Google Translate
  • UI/UX Improvements
    • Mobile app - Add the background per comment when viewing the comment/reply listing.
    • Mobile app - The comment screen when there is no comment display.

Social Connect

  • Update for compatibility with Core v5.1.10

Blog

  • Support removing storage files after deleting the Blog.
  • Integrate with the Featured Items app.

Events

  • Integrate with the Featured Items app
  • Support exporting events to ICS file
  • Support importing events to Mobile Calendar
  • UI/UX Improvements
    • Remove the text Commenting on this post has been turned off. below the event feed.
    • Display the Name Page/Group on the Event detail when the user role has no permission to view the Page/Group
    • AdminCP: Enable event ordering by default for ongoing and upcoming events.

Stories

  • UI/UX improvements
    • For the progress bar when viewing the video story.
    • The video story layout when waiting for processing.
    • The layout of the text story when viewing the detail.

  • Improve performance
  • Support parsing hyperlinks in the content when creating a new story on mobile apps.
